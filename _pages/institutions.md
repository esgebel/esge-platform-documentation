This is documentation for institutions.

## Getting started

### Activating account

To be able to access the platform, an ESGE Administrator will create an Institution Admin account for you.

The ESGE Administrator will use an email of your preference: you will receive **an invitation by e-mail** to create a password for your new account:

1. Click the link **Set up Password** in the e-mail
2. In the window that opens, enter the password you will use twice.
3. Click the button **Set Up Password**

You will be redirected to the login screen, and you can now login with your e-mail and password.

### First Login

Log in to the platform at [link](#). The first page you will see is the **Dashboard**. This gives you a birds-eye view of your institution with shortcuts to different sections of the platform.

- **Quick Actions:** shortcuts to core functions of the platform
- **Activity overview:**  an overview of the activity (training and certification sessions) that has happened the past year.
- **Pending Certification Sessions**: an overview of certification sessions that have not been finalized.
- **Upcoming Certifications**: an overview of scheduled ceritfication sessions
- **Connection Requests**: Requests from mentees to be linked with your institutiotn
- **Recent Files**: Files uploaded in your institution

### Configure the institution

To make your institution ready for operation, you need to configure several things:

- You need to create user accounts for your Mentors
- You need to configure your Institution information

As an Institution Administrator you can also perform tasks that are normally done by Mentors: for these tasks consult the [mentor focussed documentation](#).

#### Configure Institution information

1. Go to **My Account**
2. In the *Institution Information* section click **Edit**: a configuration wizard will open
3. Enter the *Basic Information*:
   - Institution Name
   - Description
   - Country
   - City and Postal Code (ZIP)
   - Address
   - Email Address
   - Phone Number and Fax Number
   - Website
4. Click **Next** and enter the *Contact Persons* information:
   - Director's name
   - Director's signature: upload an image of the signature, this will appear on the certificate
   - Chief Mentor's name
   - Chief Mentor's signature
5. Click **Next** and enter the *Institution Settings* information:
   - Add the logo
   - Add Report Test: this is optional text that will be shown on the certificate
6. Click **Save Changes**

#### Create User accounts for Mentors

As an Institution Administrator you can create user accounts for your colleagues that fullfill the role of Mentor.

1. Go to **Users**
2. Click **Add User**
3. Enter:
   - First name
   - Last Name
   - Email: an activation email to create a password will be send to this email address, so make sure it is valid!
   - Role: set the role to *Mentor*
   - Optional: set the end date of the Mentor mandate. If this end date is passed, the account will become inactive and the Mentor will not be able to use the platform anymore.
   - Specializations: select the specializations the Mentor has
4. Click **Create User**: the Mentor will receive an email to activate his/her account.

> a Mentor can only create sessions for his/her specializations. This means that a Mentor with specialization "Suturing Training" can not create training settiongs for Laparoscopy. The specialization "Certification" is mandatory to create a Certification session. A Mentor can have multiple specializations at once.
